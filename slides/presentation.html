World Models & Auxiliary Learning
for Multi-Agent Tracking

QPLEX · QPLEX-SE · QPLEX-WM2 · QPLEX-WM4 · QPLEX-WM2-SE ★

Conference Presentation  |  Multi-Agent Reinforcement Learning

MATE Environment  ·  April 2026

Abstract

We study a family of cooperative multi-agent tracking algorithms built on top of QPLEX, a duplex action-value decomposition framework. Starting from flat and hierarchical QPLEX baselines, we incrementally enrich the policy with two orthogonal ideas:

(1) Shared-Encoder auxiliary loss (SE) — a lightweight prediction head trained to reconstruct target positions from RNN hidden states, shaping internal representations;

(2) Latent World Models (WM) — an RSSM-based world model whose EMA-stable latent features augment the observation space before the recurrent policy, enabling look-ahead imagination for bootstrapping TD targets.

We further propose QPLEX-WM2-SE, which combines both ideas at complementary stages of the network, and demonstrate its potential on the MATE 4v8 benchmark, achieving a peak real coverage rate of 69.6% with HRL-QPLEX-SE.

Introduction: The MATE Environment

Multi-Agent Tracking Environment

  • N camera agents vs M evasive targets in a 2-D arena
  • Each camera controls its pan/tilt/zoom independently
  • Targets can be greedy or evasive (avoid detection)
  • Reward: fraction of targets simultaneously in camera view fields
  • Partial observability — cameras share limited messages
4v8 scenario: 4 cameras, 8 targets, complex coordination required
Coverage Rate = |tracked targets| / |total targets|

Metric: real_coverage_rate_mean — averaged across eval episodes using actual (non-fake) targets

Introduction: Challenges & Motivation

Partial Observability

Each camera sees only its local FOV. Cameras must communicate and remember to coordinate coverage.

Non-Stationarity

Evasive targets actively avoid detection, creating a dynamic adversarial environment that shifts the value landscape.

Credit Assignment

A target covered by two cameras yields no extra reward — joint coordination is crucial, individual Q-decomposition must remain faithful.

Our Approach: Augmenting QPLEX Incrementally

QMIX → QPLEX → HRL-QPLEX → +SE auxiliary → +World Model → WM2-SE ★
  • Better representation learning via auxiliary losses (SE)
  • Richer observation via latent world model features (WM2/WM4)
  • Hierarchical action decomposition via HRL framework

Method: QPLEX — Duplex Action-Value Decomposition

Key Ideas

  • DRQN backbone: GRU encodes partial obs over time
  • Individual Q-values via dueling network (V+A)
  • Duplex mixer: transforms individual Qs into joint Q satisfying IGM monotonicity
  • Weights λ computed from global state (not just obs)
Q_jt(τ,a) = λ₀(s) · V(τ) + Σᵢ λᵢ(s,aᵢ) · Aᵢ(τᵢ,aᵢ)
Qᵢ = Vᵢ(hᵢ) + Aᵢ(hᵢ, aᵢ) [dueling head]

IGM: Individual-Global-Max — greedy joint action equals argmax of joint Q

Method: HRL-QPLEX — Hierarchical Action Decomposition

HierarchicalCamera Wrapper

  • High-level policy: selects which target to track (discrete)
  • Low-level executor: moves camera toward selected target (continuous primitive)
  • Frame-skip: high-level re-plans every K steps
  • Greatly reduces action-space complexity for the learned policy
Achieves +16% coverage over flat QPLEX by decomposing coordination into target selection

Method: QPLEX-SE — Shared Encoder Auxiliary Loss

Intuition

  • GRU hidden states implicitly encode target positions — we make this explicit
  • A lightweight PredictionHead MLP maps hᵢ → predicted target positions
  • MSE auxiliary loss shapes representation without extra parameters at inference
  • Shared across all agents — maximizes data efficiency
L_SE = MSE(PredHead(h₁:ₙ), true_positions)
L_total = L_TD + α·L_SE
Key insight: operates post-RNN on hidden states — zero overhead at inference (PredHead is discarded)

Method: QPLEX-WM2 — RSSM World Model Augmentation

Latent World Model (RSSM)

  • Representation model: encodes obs + action → posterior (z_t, h_t)
  • Transition model: predicts next latent state z_{t+1} without obs (prior)
  • EMA copy of world model → stable feature extraction at train time
  • Latent features concatenated to obs before the policy GRU
  • WM trained jointly with auxiliary reconstruction loss
z_t = Encoder(obs_t, a_{t-1}, h_{t-1}) [RSSM posterior]
obs_aug = concat(obs, EMA_WM(z_t))
L = L_TD + β·L_WM_recon

Method: QPLEX-WM4 — Dreamer Imagination Rollouts

Beyond WM2: Policy-in-Loop Imagination

  • WM4 extends WM2 with an ObsDecoder head
  • Imagination rollouts: unroll H steps in latent space without env interaction
  • Bootstrapped TD targets from imagined returns (Dreamer-V2 style)
  • Evasive targets (EvasiveTargetAgent) make training harder and more realistic
z_{t+h} = TransitionModel(z_{t+h-1}, π(z_{t+h-1})) [no env]
Q_imag = R_imag + γ^H · Q_target(z_{t+H})
L = L_TD + β·L_WM + γ·L_imag_TD
EvasiveTargetAgent: targets move toward warehouse & avoid camera FOV — creates adversarial training signal

★ Proposed: QPLEX-WM2-SE — Combining Both Ideas

Key Insight: Orthogonal Augmentation

  • WM2 operates pre-RNN: enriches the input to the recurrent encoder
  • SE operates post-RNN: shapes the hidden state representations
  • No conflict — can be composed cleanly
Pipeline:
1
RSSM encodes obs → latent z_t via EMA world model
2
obs_aug = concat(obs_t, EMA_features(z_t))
3
GRU(obs_aug) → hidden hₜ
4
PredHead(hₜ) → position pred → L_SE auxiliary
5
Duplex Mixer → Q_jt → L_TD
L = L_TD + β·L_WM + α·L_SE

Experiments: Analysis & Discussion

What Works

  • HRL is the dominant improvement — decouples target selection from low-level control
  • SE auxiliary loss provides the most sample-efficient boost — reaches peak faster (0.06M vs 0.09M steps)
  • WM2/WM4 provides richer observation space — competitive with HRL-QPLEX but slower to converge

Limitations Observed

  • WM3 underperforms: more complex model but noisy gradient signals
  • WM4 imagination requires careful tensor shape alignment (B×T×A dimensionality)
  • Evasive targets in WM4 make training harder — peak 59.6% vs SE's 69.6%

WM2-SE Hypothesis

WM2 enriches what goes into the GRU → better temporal compression

SE refines what comes out of the GRU → better spatial representations

Together they address complementary bottlenecks — expected to be strictly additive.

Expected: Coverage(WM2-SE) ≈ Coverage(SE) + ΔCoverage(WM2/QPLEX)
Future work: run WM2-SE training & compare against both branches; explore WM4+SE for full Dreamer + representation auxiliary combination

Conclusion

Summary

  • Presented a systematic family of QPLEX extensions for multi-agent camera tracking
  • HRL wrapper is the most impactful single component
  • SE auxiliary loss shapes representations cheaply — best performing variant at 69.6%
  • World Models (WM2/WM4) provide richer inputs at cost of increased complexity
  • Proposed novel QPLEX-WM2-SE combining orthogonal augmentations
Key result: HRL-QPLEX-SE achieves 69.6% real coverage on MATE 4v8, a +41 pp improvement over QMIX baseline
